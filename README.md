## 🏰 **Guided Tour of the Colomares Monument**
This repository contains the official website for the Guided Tour of the Colomares Monument, created by Carla and Alejandra.
The purpose of this project is to offer visitors a clear, accessible, and multilingual experience, allowing them to view all tour information by scanning a QR code located at the monument.

<h3 style="background:#f0f0f0; padding:8px;">📌 What does this website offer?</h3>
When scanning the QR code, visitors are taken to a page where they can select their preferred language from a wide variety of options.
Each language provides access to a complete PDF that includes:

- A general introduction to the Colomares Monument

- The 17 points of the guided tour

- A detailed and extended explanation in Point 17

- The interface displays flags and country names, each with a button that opens or downloads the corresponding PDF.

### 🌐 Available languages
The website includes a broad selection of European and Asian languages, such as:

- Spanish, English, French, German, Italian, Portuguese, Croatian, Polish, Danish, Slovak, Czech, Lithuanian, Hungarian, Romanian, Greek, Bulgarian, Serbian, Norwegian, Dutch, Latvian, Swedish, Finnish, Russian, Turkish, Hindi, Chinese, Korean, Thai, Japanese, and Arabic.

The exact list may vary depending on the PDFs included in the repository.

### 🎨 Design and user experience
Simple, visual, and universal interface

Language selection through flags + country names

Direct access to each PDF

Fully responsive design, optimized for mobile devices

Fast and intuitive navigation

### 🛠️ Technologies used
HTML5 — website structure

CSS3 — visual design and responsive layout

JavaScript (optional) — language selector interactions

Git & GitHub — version control and collaboration

### 🚀 How to view the project
Clone the repository:

git clone https://github.com/usuario/tour-colomares.git

Open the main file:

index.html

Select a language and open the corresponding PDF.
